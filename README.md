# AIPACScan

A parody database-style website tracking AIPAC funding (inspired by SomaliScan). For entertainment purposes only.

## Features

- **Database Interface** - Browse and search entities receiving AIPAC funding
- **Real-time Stats** - Track total entities and documented funding
- **Multiple Categories** - Politicians, Organizations, Donors, and Lobbyists
- **Add Entities** - Upload photos and add new entries to the database
- **Search Functionality** - Fast search with keyboard shortcut (⌘K)
- **Red Theme** - Professional dark mode with red accents
- **Persistent Storage** - LocalStorage saves all added entities
- **Responsive Design** - Works on all devices

## Pages

1. **Home/Map** - Overview with statistics and categories
2. **Database** - Browse all tracked entities
3. **Political** - Analysis by political party
4. **Analysis** - Detailed funding patterns (coming soon)

## How to Use

### Viewing the Database
1. Open `http://localhost:8000` in your browser
2. Browse the default entities or search for specific ones
3. Click any entity to view detailed information

### Adding Entities
1. Click "+ Add Entity" button
2. Upload a photo (optional)
3. Enter name and select type
4. Submit to add to database
5. View in the database instantly

### Search
- Use the search bar or press `⌘K` (Mac) / `Ctrl+K` (Windows/Linux)
- Filter by name or type
- Results appear instantly

## Technologies

- Pure HTML5, CSS3, JavaScript (no frameworks!)
- LocalStorage for data persistence
- CSS Grid & Flexbox for layouts
- Modern dark theme with CSS variables
- Google Fonts (Inter)

## Data

All amounts are randomly generated between $10K - $60M. The site includes 16 default entities covering politicians, organizations, donors, and lobbyists. All data is fictional and for parody purposes only.
